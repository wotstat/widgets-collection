Виджет для отображения рекомендуемого оборудования и полевой модернизации для каждого танка от игрока Sh0tnik.

С полным списком вы можете ознакомиться на сайте [shotnik.pro](https://shotnik.pro/equipment/){target=_blank}

> [!NOTE]
> Если у вас нет бонового или экспериментального оборудования, ставьте его аналоги любого цвета. Также не забывайте, что для штатного оборудования есть соответствующие ролевые слоты.

### Текущий выбор
- Зелёная галочка – у вас уже установлена верная модернизация
- Красный крестик – у вас установлена неверная модернизация
